# Lore Tracker

A web application to track and manage lore entries for various fictional universes.

## Features

- Add, edit, and delete characters.
- Manage relationships between characters.
- Manage locations.
- Track events and timelines.
- Search and filter lore entries.

## Technologies Used

- PHP
- Symfony Framework
- Twig Templating Engine
- Doctrine ORM
- Bootstrap for styling
